# Road Project #1 - Documentação

Estrutura do Projeto

  - index.html: Página principal com estrutura HTML do currículo
  - style.css: Arquivo de estilos com:
  - Layout em formato de documento Word
  - Cores e tipografia profissional
  - Espaçamento adequado entre seções
  - script.js (se existir): Contém funcionalidades JavaScript


Principais Estilos

- Gradiente roxo para rosa no background
- Container principal com 21cm de largura (formato A4)
- Fontes similares ao Word (Calibri/Arial)
- Espaçamento entre parágrafos e seções


Como Usar

1. Abra `index.html` em qualquer navegador
2. Para editar, modifique:
   - Conteúdo: index.html
   - Estilos: style.css
   - Comportamento: script.js


Personalização

Você pode alterar:
- Cores no arquivo CSS
- Conteúdo no HTML
- Adicionar novas seções conforme necessário
